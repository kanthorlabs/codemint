# User Story 4.10: YOLO Mode Delegation UI (Supports EPIC-02)

* **As a** Developer using the UI (TUI/CUI),
* **I want to** see a `[Delegate to Auto Approval]` action button next to Epics and User Stories,
* **So that** I can trigger the autonomous execution mode defined in the Brainstorming phase.
* *Acceptance Criteria:*
    * The TUI Right Panel allows selecting an Epic/Story and pressing a hotkey to toggle Auto-Approval.
    * The CUI inline keyboard for an Epic/Story includes a `[🤖 Auto-Approve]` button.
