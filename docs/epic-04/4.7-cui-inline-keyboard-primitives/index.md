# User Story 4.7: CUI Inline Keyboard Primitives

* **As a** Developer using Telegram/Slack,
* **I want** UI prompts to render as native Inline Keyboards,
* **So that** I can approve or select options with a single tap.
* *Acceptance Criteria:*
    * Single-select prompts render as standard inline buttons.
    * Multi-select prompts render as dynamic keyboards that update with a `✅` emoji upon clicking, followed by a final `[Submit]` button to dispatch the response.
