# User Story 4.6: CUI Pull Commands

* **As a** Developer using Telegram/Slack,
* **I want** to use slash commands to pull information on demand,
* **So that** I can check the system state without having it forced on my screen.
* *Acceptance Criteria:*
    * `/tasks` returns the current Epic -> Story -> Task hierarchy list.
    * `/status` returns the currently active task.
