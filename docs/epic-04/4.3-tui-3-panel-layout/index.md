# User Story 4.3: TUI 3-Panel Layout

* **As a** Developer using the terminal,
* **I want** a highly functional 3-panel layout,
* **So that** I can see the chat log, my input, and the project context simultaneously.
* *Acceptance Criteria:*
    * Left Panel (Top) renders the Chat Log.
    * Left Panel (Bottom) renders a multi-line input box and a sticky status bar displaying Project Name, Git Branch, and local time.
    * Right Panel renders the Context Window (Epic -> Story -> Task list with color-coded status).
